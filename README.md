# Mission 0 - Mulan Static Web Application
 
## Project Overview
 
This is a static website project built with HTML, CSS, and JavaScript, representing foundational 
web development skills. The project showcases a themed website with responsive design and interactive 
JavaScript functionality.
 
## Features
 
- **Responsive Design**: Mobile-friendly layout that adapts to different screen sizes
- **Interactive Elements**: JavaScript-driven functionality for enhanced user experience
- **Modern Styling**: CSS3 features including animations, transitions, and flexbox/grid layouts
- **Semantic HTML**: Proper HTML structure for accessibility and SEO
- **Cross-browser Compatible**: Works across modern browsers
 
## Technologies Used
 
- HTML5
- CSS3
- JavaScript (ES6+)
- Responsive Design
- Flexbox/CSS Grid
 
## Project Structure
 
```
mission-0-lv5-Mulan_staticWEBAPP/
├── index.html          # Main HTML file
├── styles.css          # Main stylesheet
├── script.js           # JavaScript functionality
├── images/             # Image assets

```
 
## Getting Started
 
### Installation
 
1. Clone the repository:
```bash
git clone https://github.com/Lharry95/mission-0-lv5-Mulan_staticWEBAPP.git
cd mission-0-lv5-Mulan_staticWEBAPP
```
 
2. Open in browser:
```bash
# Open index.html in your web browser
open index.html
```
 
### Usage
 
Simply open the HTML file in a web browser. The website is fully functional and ready to use without 
any additional setup or dependencies.
 
## Key Features Demonstrated
 
- **HTML5 Semantic Elements**: Proper use of <header>, <nav>, <main>, <section>, <footer>
- **CSS3 Styling**: Advanced styling including transitions, animations, and modern layout techniques
- **JavaScript Interactivity**: DOM manipulation, event listeners, and dynamic content updates
